项目结构说明
server.cpp：服务器端实现。基于协程库构建了一个简单的回声（Echo）服务器，监听本地 8080 端口。可通过浏览器或 HTTP 客户端访问 http://localhost:8080，服务器会返回包含请求内容的响应。
client.cpp：客户端实现。模拟浏览器行为，向本地服务器发起 HTTP 请求，用于测试服务器的并发处理能力与响应正确性。
其他 .cpp 文件：协程库的核心模块，实现了用户态协程的调度、上下文切换、I/O 事件驱动等功能，为服务器提供高并发支持。
可执行程序
build\server：当前实现为一个简易 HTTP 回声服务，可用于验证协程调度与非阻塞 I/O 的正确性。
build\client：轻量级客户端程序，可批量发起 HTTP 请求
